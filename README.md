# Flask App

## Features
- Dictionary from MySQL database
- Scraping and adding dictionaries from Multitran.com
- CRUD functionality
- Search form
- Translation hints

## Tech Stack
- Flask
- MySQL
- Selenium
- Pandas
- NLTK
